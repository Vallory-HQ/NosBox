# NosBox
Vagrant Box para Nosotros

## Specs
| Ubuntu LTS | Mysql Version | Php Version | NosBox Version | Branch
| -----------| -----------   | ----------- | -----------    | -----------
| 20.04      | 5.7           | 7.4         | 1.0            | `master`

## Set up and running
```
$ git clone https://github.com/Vallory-HQ/NosBox.git ~/NosBox
```
```
$ cd ~/NosBox
$ vagrant up
```
